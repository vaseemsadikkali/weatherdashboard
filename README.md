# 🌦 Weather Dashboard - Pro Edition

A modern and responsive weather dashboard built using HTML, CSS, and JavaScript. This project provides real-time weather information, forecasts, air quality monitoring, and interactive weather insights through a clean and user-friendly interface.

## 🚀 Features

* Real-time weather information
* 5-Day Weather Forecast
* 24-Hour Weather Timeline
* Air Quality Index (AQI) Monitoring
* Weather Alerts & Emergency Notifications
* AI-Powered Weather Insights
* Dark/Light Theme Toggle
* Location-Based Weather Search
* Sunrise & Sunset Information
* Responsive Design for Desktop and Mobile

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API
* Git & GitHub
